Retail Data Visualization using R Description

This project uses R to analyze and visualize retail business data. It creates different plots to understand Sales, Profit, Customer Segment, and Region.

Requirements

R

RStudio

Packages used:

ggplot2

dplyr

tidyr

Dataset

File name: 6.retail_business.csv

The dataset should be in the same folder as the R script.

Visualizations Created

Parallel Coordinate Plot – compares Sales and Profit by Customer Segment

Bubble Chart – shows Sales vs Profit with bubble size based on Sales

Trellis Plot – Sales vs Profit displayed by Region

How to Run

Open the R script in RStudio

Set the working directory to the dataset location

Run the code

Output

The code generates graphical plots for better understanding of retail performance.
